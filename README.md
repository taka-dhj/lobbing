# 予約管理システム

テニスコート・宴会場の予約管理システムです。

## 機能

- 月毎の売上予定の表示
- 予約の追加・編集・削除
- 合計金額の自動計算
- ローカルストレージによるデータ永続化
- 直感的なUI/UX

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```

## 使い方

1. 「新しい予約を追加」ボタンをクリックして予約を追加
2. 月毎にグループ化された予約一覧で売上を確認
3. 各予約の「編集」ボタンで予約内容を変更
4. 「削除」ボタンで予約をキャンセル

データはブラウザのローカルストレージに自動保存されます。
